# Simple-linear-regression
This project demonstrates Simple Linear Regression to predict salary based on years of experience. It covers data analysis, visualization, model training, and evaluation using Python and Scikit-learn. The trained model is saved for future predictions, showing a complete beginner-friendly ML workflow.
# Simple Linear Regression Project

# Overview
This project implements **Simple Linear Regression (SLR)** to predict salary based on years of experience. It is a supervised machine learning algorithm used to model the relationship between two variables.

The project demonstrates the complete machine learning workflow including:
- Data preprocessing
- Model training
- Model evaluation
- Data visualization
- Model saving and loading
